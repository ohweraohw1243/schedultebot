# бот-календарь

пет-проект,  который я написал, чтобы все расписание было в одном месте. Я учусь в универе в НГТУ и параллельно подрабатываю репетитором. Из-за этого расписание всегда лежит в двух разных местах: у вуза на сайте (статичное), а у учеников в CRM-системе "МойКласс" (постоянно меняется, бывают отмены и переносы за пару часов).

Поэтому я решил сделать бота, который сам лезет по всем моим сайтам, собирает расписание, сохраняет его к себе в базу и выдает мне удобную ссылку для виджетов типа Structured. А если кто-то из учеников внезапно перенес занятие — бот напишет мне об этом в телегу.
